# Project Management Dataset

This repository contains the **Project Management Dataset**, a structured dataset suitable for analyzing project performance, resource utilization, strategic impact, and profitability in project management contexts. The dataset was originally published on **Kaggle** by hosammhmdali and includes detailed information for 99 projects across multiple business and operational dimensions. ([Kaggle Dataset](https://www.kaggle.com/datasets/hosammhmdali/project-management-dataset?utm_source=chatgpt.com))

## Dataset Overview

- **Number of records:** 99 projects  
- **Number of columns:** 16  
- **File format:** CSV  
- **Source:** Kaggle — _Project Management Dataset_ ([Kaggle Dataset](https://www.kaggle.com/datasets/hosammhmdali/project-management-dataset?utm_source=chatgpt.com))

 Download the dataset from Kaggle:

## Columns and Descriptions

| Column | Description |
|--------|-------------|
| **Project Name** | Name of the project |
| **Project Description** | Brief description of the project |
| **Project Type** | Type of project (Income Generation, Cost Reduction, Process Improvement, etc.) |
| **Project Manager** | Manager responsible for the project |
| **Region** | Geographical region of the project |
| **Department** | Responsible department |
| **Project Cost** | Project cost |
| **Project Benefit** | Expected benefit or profit |
| **Complexity** | Project complexity level (Low, Medium, High) |
| **Status** | Project status (Completed, In-Progress, On-Hold, Cancelled) |
| **Completion%** | Project completion percentage |
| **Phase** | Project phase (Explore, Develop, Plan, Implement, Measure) |
| **Year** | Project execution year |
| **Month** | Project start month |
| **Start Date** | Project start date |
| **End Date** | Project end date |
